Design Principles

Near-monochrome ink

Almost no saturated colour. Light theme is near-black on white, dark theme inverts. Semantic colours are desaturated earth tones.

Printed, not digital

Sharp corners (2px radius), no shadows, heavy typographic borders, small-caps metadata, serif headlines. The UI reads like a broadsheet.

British English

All user-facing text uses British spelling: favourite, colour, organisation. CSS class names remain American (industry convention).

Content first

Reader view uses Georgia at 1.125rem/1.8 line-height with hanging punctuation and optical kerning. The UI disappears when reading.
